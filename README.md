# Ecommerce ML Project

## Description
Projet de machine learning pour prédire les dépenses annuelles des clients à partir de données e-commerce.

## Étapes
1. Prétraitement des données (`preprocessing.py`)
2. Entraînement du modèle (`train.py`)
3. Évaluation (`evaluate.py`)
4. Prédiction via API (`api/main.py`)
